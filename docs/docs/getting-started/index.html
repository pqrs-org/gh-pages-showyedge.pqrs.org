Getting Started

System requirements

  • macOS 10.9 or later

Installation

  1. Download dmg file from here.
  2. Open a downloaded dmg file.
  3. Move ShowyEdge icon into Applications icon.
  4. Open ShowyEdge from Launchpad.
  5. Preferences will be opened. Change values if needed and then close the window.
